# Inventory Management System (C# Console Application)

```

## Technologies Used
- C#
- .NET 6+/7+/8+
- Console Application
- Object-Oriented Programming

## How to Run
1. Clone the repository
```bash
git clone <your-repo-url>
```

2. Open project in Visual Studio / VS Code

3. Run the application
```bash
dotnet run
```

## Sample Menu
```bash
=== Inventory Management ===
1. Add Product

3. Search Product
4. Delete Product
5. Exit
